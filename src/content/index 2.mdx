# Brand System Oversize

Bem-vindo ao sistema centralizado de diretrizes de marca da Oversize.

Este é o ponto de partida para entender como a Oversize comunica, se posiciona, se apresenta e entrega valor.

## O que você encontra aqui

**Voz e Identidade** — Como a Oversize fala, seus princípios de comunicação e identidade visual.

**Metodologia** — A Estratégia Estruturada em 12 passos que estrutura tudo que a Oversize faz.

**ICP e Negócio** — Quem é o cliente ideal, qual é o inimigo comum e como oferecemos valor.

**Territórios de Conteúdo** — Os pilares temáticos e estratégias de conteúdo da marca.

**Diretrizes Visuais** — Paleta de cores, tipografia, logo e padrões visuais.

**Formatos e Roteiros** — 7 formatos B2B prontos para usar em comunicações e campanhas.

## Como usar

Use a navegação lateral para explorar cada seção. Cada página contém informações estruturadas e exemplos práticos.

Você pode buscar por palavras-chave na barra de busca para encontrar o que precisa rapidamente.

## Próximos passos

Recomendamos começar por **Voz e Identidade** para entender os princípios fundamentais, depois explorar a **Metodologia** para ver como tudo se conecta.
